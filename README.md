# Cobweby Stone Brick
#### ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Adds a new block the **Cobweby Stone Brick**,
this block is a trap block which when stood on for too long or landed on,
breaks and reveals a cobweb block inside it which the player/mob gets stuck in
<br>
